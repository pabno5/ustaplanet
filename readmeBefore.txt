Librerías usadas:
	react
	three
	sass
	@mui/material
	pg
	node
	cors


Ejecutar el siguiente comando para instalar las librerías:
	npm install react three sass @mui/material pg node cors


Usando el aplicativo XAMPP, agregar la base de datos que se encuentra en el archivo:
	src/sql/usuarios.sql


Encender la base de datos y jecutar el comando:
	node server.js
	npm run dev
